Ejercicios de Practico 7

Objetos |Array asociativos

Un click en boton de ejercicio para abrir y cerrar el ejercicio.

Crear la interfaz y la funcionalidad para almacenar datos de personas(nombre,edad,nacionalidad).

Crear un botón que al ser presionado arme una tabla con todos los datos de personas disponibles.

Crear otro botón que arme una nueva tabla con todos los datos de las personas mayores de edad.




Un cine nos pide crear una aplicación para almacenar películas. De cada película se quiere guardar el nombre(texto), el año (numero), el género(comedia,drama o ciencia ficción), la cantidad de votantes (número) y el total de puntos totales recibidos(número).






En un local de venta de guitarras se venden tres tipos de guitarras: clásicas, electricas y electroacústicas. De cada tipo existe un único modelo.Se solicita generar una aplicación para la gestion de ventas de productos.

Precios: Clasica 300, Electrica 450, Electroacusticas 500.

Una empresa de venta de celulares quiere llevar registro de sus ventas. Para cada venta se ingresa marca(Samsung, Sony o LG), modelo(texto), precio(numero) y cantidad de unidades(número).






Crear una interfaz HTML(solamente el contenido de dentro del body) con todos los elementos necesarios para solicitar datos de zapatos. De cada zapato se debe solicitar marca, talle, color y si es nacional o importado.






Dada una pagina HTML que contiene un elemento "contenido", escriba una funcion en javascript que dibuje un formulario dentro del elemento en el momento de ser cargada la pagina.
